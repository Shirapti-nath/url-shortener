# URL Shortener Backend

A backend application built with Node.js, Express, PostgreSQL, and Prisma.

## Features

- Create Short URLs
- Custom Short Codes
- URL Validation
- Redirect Original URLs
- Click Tracking
- Analytics API

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Prisma ORM

## API Endpoints

POST /api/shorten

GET /:shortCode

GET /api/analytics/:shortCode
